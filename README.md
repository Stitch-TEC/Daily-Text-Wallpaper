## ‚ú® Features

* **Automatic Fetching:** Pulls the current day's text and scripture directly from WOL in 7 different languages (English, Spanish, Japanese, French, Portuguese, German, and Italian).

    * **Dynamic Overlay:** Apply a colored tint (custom color picker) with adjustable opacity to ensure text is legible over any photo.

    * **Date Labels:** Toggle between a floating header, inline text, or no date at all.

2.  **Upload Photo:** Click **üì∏ Upload Photo** to pick a background image from your device.
